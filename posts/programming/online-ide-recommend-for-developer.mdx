## 들어가는 글

안녕하세요.😊 오늘은 따로 개발 환경을 구성할 필요 없이 온라인상으로 개발할 수 있도록 도와주는 온라인 ide 4가지를 추천드리고자 합니다.

## ideone

<br/><br/>
<img src="/ide-one.webp" alt="captured display about ideone" width={500} height={300} />
<br/>
가장 첫번째로 [ideone](https://ideone.com/)입니다.
ideone은 굉장히 심플하고 직관적이라고 생각이 드는데요.
언어를 테스트 하기위한 최소한의 환경을 제공해 주는 느낌이 듭니다.
지원하는 언어는 Bash, C, C++, Java, Objective-C, PHP, Python, Swift, Javascript 등이있습니다.
<br/><br/>

## codepen

<br/><br/>
<img src="/code-pen-1.webp" alt="captured display about codepen" width={700} height={300} />
<br/>

두번째로 [codepen](https://codepen.io/)입니다.
codepen은 특히 html, javascript, css를 온라인에서 작성하기에 최적화 된 느낌이 있습니다. 프론트엔드에 관심이 많다면 codepen에 다른 사람들이 업로드 해둔 다양한 예제를 이미 많이 접해보셨을 수도 있다는 생각이 듭니다.

<br/><br/>
<img src="/code-pen-2.webp" alt="captured display about codepen" width={700} height={300} />

실제로 코드를 작성하는 환경은 위의 이미지와 같은데요. 실시간으로 코딩의 결과가 빈 공백화면에 바로 반영 되어 내가 작성중인 코드의 결과를 바로바로 확인해 볼 수 있다는 것이 가장 큰 장점입니다.


## 구름 IDE

<br/><br/>
<img src="/goorm-ide-2.webp" alt="captured display about goorm ide" width={700} height={300} />
<br/>

세번째로 [구름 IDE](https://ide.goorm.io/)입니다.
국내 회사인 구름에서 만든 IDE로 종량제 서비스 입니다만, 처음에 일정 금액은 무료로 지원을 해줍니다.
UI가 예쁘고 개발 환경 & 언어의 옵션이 굉장히 세밀하게 나누어져 있다는 것이 강점입니다.
<br/>

<img src="/goorm-ide-1.webp" alt="captured display about ideone" width={300} height={300} />

<br/>
사진에서 처럼 스택, OS, 성능등을 다양한 옵션에서 고를 수 있습니다.
스택별 템플릿이 이미 구성 되어있는 것들도 많아서 바로 가져다 쓸 수 있게 되어있습니다.
React, NVIDIA Tesla T4, .Net, Arduino, Spring, Tensorflow, Vue, React 등 다양한 스택이 있습니다.
깃허브에서 불러오기도 가능하다고 합니다.

## ryugod
<br/><br/>
<img src="/goorm-ide-2.webp" alt="captured display about goorm ide" width={300} height={300} />
<br/>

마지막으로 [RyuGod](https://www.ryugod.com/pages/ide/bash)입니다.
bash, typescript, java, html, c, react 등등 입맞에 맞게 언어를 고를 수 있습니다.
제일 큰 장점은 언어를 선택하면 바로 왼쪽에 예제를 제공해준다는 것입니다.
예제를 누르면 바로바로 코드가 보여서 처음 언어를 배우는 분들이 접하기 좋을 것 같다는 생각이 듭니다.
